## 1.5M波特率USB转UART串口 1.0.0


用于日常的调试等，通过串口可以将产品和电脑连接，从而控制产品。


![](/assets/images/串口版.jpg) ![](/assets/images/串口版-背.jpg)

## 串口引脚

| 名称 | 功能 |
| ---- | ---- |
| 3.3V | 给产品供电 |
| 5V   | 给产品供电 |
| GND  | 地 |
| RX   | UART的RX（接产品的TX） |
| TX   | UART的TX（接产品的RX） |
| IO0  | IO口（一般不使用） |
| RST  | 复位（一般不使用） |
| DSR  | 下压检测（一般不使用） |
| CTS  | 数据流控（一般不使用） |
